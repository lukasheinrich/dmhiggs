Recast calculating fiducial efficiencies for Mono-Higgs models

Introduction

This is a accompanying web service to the Higgs+MET (mono-Higgs) analysis carried out by the ATLAS experiment: ATL-COM-PHYS-2014-1564. As many dark matter models may produce this collider signature, this web service provides an interface to calculate the fiducial efficiencies for a user-provided models. The user must upload generated Monte Carlo samples in the Les Houches Event format.

Additionally this serves as a test-bed for the RECAST project.
